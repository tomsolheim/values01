# Acceptance: Project Start

## Criteria

- The application boots on Laravel 13.
- Livewire 4 is installed.
- Bootstrap 5 is available through Vite.
- The project includes spec folders for product vision, features, acceptance criteria, and decisions.
- The root page renders a simple Bootstrap layout.

## Suggested Tests

- A feature test confirms the root page returns HTTP 200.
- A feature test confirms the root page contains the application name.
